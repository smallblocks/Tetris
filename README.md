# Tetris for SATSCADE

A classic Tetris clone for the SATSCADE arcade on StartOS.

## Features

- Classic Tetris gameplay
- Keyboard and touch controls
- Score tracking
- Level progression
- Retro arcade aesthetic

## Controls

**Keyboard:**
- ← → : Move piece
- ↑ : Rotate
- ↓ : Soft drop
- Space : Hard drop
- P : Pause

**Touch:**
- Swipe left/right: Move
- Tap: Rotate
- Swipe down: Drop

## Installation

Requires SATSCADE to be installed first. Install from the Start9 marketplace or sideload the `.s9pk` file.

## Building

```bash
# Requires start-cli
make build
```
